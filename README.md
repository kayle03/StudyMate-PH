# StudyMate PH 🇵🇭
A Bayanihan-driven, interactive learning companion platform designed to elevate the academic journey of Filipino students. Organize, collaborate, share review materials, and track academic milestones alongside your classmates in real-time.

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18 or higher recommended) installed on your system.

### 2. Open the Project
1. Download the ZIP file from Google AI Studio.
2. Extract the archive.
3. Open **VS Code** (`File > Open Folder`) and select the extracted folder.

### 3. Install Dependencies
Open standard terminal window in VS Code (`Ctrl + ~` / `Cmd + ~`) and run:
```bash
npm install.

### 4. Run Development Server
Boot up Vite's lightning-fast development server:
code
Bash
npm run dev
### 5. Open in Browser
Once active, hold Ctrl and select the local address outputting in your terminal (usually http://localhost:3000) to access the fully functional, real-time sync study portal!
